1. Create a folder named auction.

2. Create a auction.cvs file inside auction folder with specific format (as shown in the sample.zip).

3. Create a folder named poster_photo inside auction and place all the images to be uploaded. Note: All the image name should match the name as in the auction.csv file.

4. Zip the auction folder and upload it. Zip size should not exceed 20 mb.

5. Mandatory fields  - Poster Title, Poster Description, Poster Images, Size, Genre, Decade, Country, Condition, Asked Price.
